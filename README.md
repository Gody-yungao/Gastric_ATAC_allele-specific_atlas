# Gastric_ATAC_allele-specific_atlas
This repository contains the codebase used in the study **"An integrative chromatin accessibility atlas of human gastric tissue links regulatory genetic variants to gastric cancer susceptibility"**, covering both main analysis and figure generation scripts.

The project aims to construct a chromatin accessibility landscape of human gastric tissue and identify the individual characteristics driving its inter-individual variation. Furthermore, it seeks to uncover genetic variants that modulate gastric chromatin accessibility and, by linking these to gastric cancer susceptibility, ultimately elucidate the genetic regulatory mechanisms mediating gastric cancer risk through altered chromatin accessibility.
